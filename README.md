# Balloon_Text_Maker

## 📝 概要

Balloon_Text_Maker は、Blender 上で文字を立体化し、厚み付け・細分化・リメッシュを簡単に実行できる軽量アドオンです。  
「文字を風船のように膨らませたい」「立体文字を素早く作りたい」といった用途で、モデリング準備工程を大幅に短縮します。

---

## 主な機能

### 1. テキスト作成

- 入力した文字列からテキストオブジェクトを自動生成
- 任意のフォントを選択可能

### 2. メッシュ化 & 厚み付け

- テキストをメッシュへ変換
- **SOLIDIFY** による厚み付け
- **BEVEL** によるエッジの丸み付け

### 3. 細分化（＋スムーズリメッシュ）

- メッシュを指定回数細分化
- 任意で Remesh（SMOOTH モード）を追加
  - Octree Depth
  - Scale
  - Voxel Size
  - 分離断片削除オプション
  - スムースシェーディング

### 4. 一括処理ボタン

- 「Create → Meshify → Subdivide」を **ワンクリック**で実行

---

## インストール方法

1. このアドオンフォルダを zip 化
2. Blender の **Edit → Preferences → Add-ons**
3. **Install…** を押して zip を選択
4. 「Balloon Text Maker Lite」を有効化

---

## 使い方

1. **N キー → BalloonTextLite タブ**を開く
2. テキスト・フォント・厚み・細分化などを設定
3. 以下の順にボタンを押すだけ
   - ① **Create Text**
   - ② **Meshify & Thicken**
   - ③ **Subdivide Mesh**
4. リメッシュを使いたい場合は **Use Remesh** を ON

---

## 🔧 プログラム処理フロー

1. 文字列とフォントを元にテキストオブジェクトを生成
2. テキストをメッシュへ変換し、厚み付け・ベベルを自動適用
3. メッシュを細分化し、必要に応じてリメッシュを追加
4. 全工程をまとめて実行可能なワークフローを提供

---

## このアドオンが解決する課題

- 立体文字作成の「追加 → メッシュ化 → 厚み付け → 細分化」の繰り返しを大幅短縮
- リメッシュの細やかな調整を UI 化し、アーティストが試行錯誤しやすい環境を提供
- 初心者でも簡単に使えるステップ構成で、迷わず立体文字を生成可能
